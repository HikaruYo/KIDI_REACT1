~Functional Components

Functional Components adalah Javascript Function yang menerima Input properti yang menghasilkan Output berupa HTML (JSX)


~Class Components

Class Components hampir sama dengan Functional Components, tetapi Class Components dapat menyimpan sebuah Private Internal State (dapat menyimpan informasi private pada Component tersebut dan menggunakannya sebagai UI)


~Props

Props digunakan untuk menerima Value dan mengembalikan Value untuk ditampilkan ke layar
